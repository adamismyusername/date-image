Frequently Asked Questions

Everything you need to know about DateImage Pro

The date images automatically update daily at midnight UTC. Your embedded images will always show the current date without any manual intervention required. The update process is handled by our automated GitHub Actions workflow.

Currently, all date images are provided in PNG format. PNG offers the best quality for text rendering and supports transparency, making it perfect for overlaying on different backgrounds. The images are optimized for web use to ensure fast loading times.

Reach out to Adam G with the specific requirements for your desired font, color, or size. He'll be able to assist you in generating a custom image that meets your needs.

Simply use the "Copy HTML Code" button to get a ready-to-paste HTML img tag, or copy the direct URL and use it in your existing img tags. The image will automatically update daily. You can also use it in CSS as a background image or in any platform that accepts image URLs.

Date images provide consistent rendering across all platforms and browsers, eliminating font compatibility issues. They're perfect for email signatures, newsletters, and platforms with limited CSS support. Plus, they automatically update without requiring any server-side code or JavaScript.
